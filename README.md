# skapa-en-inloggningssida
Inlämningsuppgift

Min reflektion:

Programmet börjar med att kontrollera att användaren är verifierad sen tidigare, det vill säga att användaren har ett objekt lagrat i localStorage kallat "isVerified". Jag har valt att använda localStorage som en lagrad boolean i denna uppgiften.
Jag är van sen tidigare att programmera i C# och C++ där funktioner deklareras utanför andra funktioner. Därför har jag försökt att använda fördelarna med att delarera funktioner i funktioner i 'ApplyWelcomeScreen()' där jag använder 'addEventListener()' till 'logoutBtn'.
Jag försökte använda funktioner så mycket som möjligt för läsbarhetens skull, det hjälpte mig att lösa uppgiften ett steg i taget. Koden talar annars för sig själv.

Jag har valt att inte göra någon styling då det var valfritt. Kalla mig lat, men skönheten ligger på insidan även här. Dessutom kommer det fler möjligheter att skryta om CSS-kunskap senare.
